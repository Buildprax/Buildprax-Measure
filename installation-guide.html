BUILDPRAX MEASURE PRO

Installation Guide for macOS

⚠️ Important: If you see "app is damaged" after downloading, this is normal macOS security behavior. Follow the steps below to fix it.

Quick Installation (Recommended)

Step 1: Download and Open

After downloading, locate the BUILDPRAX MEASURE PRO-1.0.0-arm64.dmg file in your Downloads folder.

Step 2: Remove Security Quarantine

Open Terminal (Press ⌘ + Space, type "Terminal", press Enter) and run:

xattr -d com.apple.quarantine ~/Downloads/"BUILDPRAX MEASURE PRO-1.0.0-arm64.dmg"

This removes macOS security restrictions from the downloaded file.

Step 3: Mount and Install

Double-click the DMG file to mount it, then drag "BUILDPRAX MEASURE PRO" to your Applications folder.

Step 4: Remove App Quarantine

In Terminal, run:

xattr -d com.apple.quarantine "/Applications/BUILDPRAX MEASURE PRO.app"

Step 5: Launch the App

Find "BUILDPRAX MEASURE PRO" in your Applications folder and double-click to launch.

First Launch: If you see "Developer cannot be verified", right-click the app and select "Open", then click "Open" in the security dialog.

✅ Success! The app should now open normally. You'll only need to do this once.

Troubleshooting

"App is damaged and should be moved to the Trash"

This is a macOS security message. Follow Steps 2 and 4 above to remove quarantine flags.

"Developer cannot be verified"

Right-click the app in Applications → Select "Open" → Click "Open" in the dialog. This only needs to be done once.

Permission Denied Error

If Terminal shows permission errors, try adding sudo before the commands (you'll be asked for your password):

sudo xattr -d com.apple.quarantine ~/Downloads/"BUILDPRAX MEASURE PRO-1.0.0-arm64.dmg"

System Requirements

  • macOS: 10.15 (Catalina) or later
  • RAM: 4GB minimum, 8GB recommended
  • Storage: 200MB free space
  • Architecture: Apple Silicon (M1/M2/M3) or Intel

Getting Started

  1. Launch the app from your Applications folder
  2. Create a new project using the "New Project" button
  3. Upload a PDF drawing using "Upload Drawing"
  4. Set the scale by clicking "Set Scale" and measuring a known distance
  5. Start measuring using Length, Area, or Count tools
  6. Export to Excel when finished
