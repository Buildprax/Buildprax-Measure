BUILDPRAX MEASURE PRO

Installation Guide (Windows & macOS)

✅ Approved software: Windows installs from the Microsoft Store. macOS downloads are Apple-signed and notarized (approved by Apple).

Windows (Microsoft Store)

Step 1: Install from the Microsoft Store

On Windows 10/11, install BUILDPRAX MEASURE PRO from the Microsoft Store, then launch it from the Start Menu.

macOS (Recommended)

Step 1: Download and Open

After downloading, locate the BUILDPRAX MEASURE PRO .dmg file in your Downloads folder (the filename may include a version number).

Step 2: Mount and Install

Double-click the DMG file to mount it, then drag "BUILDPRAX MEASURE PRO" to your Applications folder.

Step 3: Launch the App

Find "BUILDPRAX MEASURE PRO" in your Applications folder and double-click to launch.

✅ Success! The app should now open normally.

System Requirements

  • macOS: 10.15 (Catalina) or later
  • RAM: 4GB minimum, 8GB recommended
  • Storage: 200MB free space
  • Architecture: Apple Silicon (M1/M2/M3) or Intel

Getting Started

  1. Launch the app from your Applications folder
  2. Create a new project using the "New Project" button
  3. Upload a PDF drawing using "Upload Drawing"
  4. Set the scale by clicking "Set Scale" and measuring a known distance
  5. Start measuring using Length, Area, or Count tools
  6. Export to Excel when finished
