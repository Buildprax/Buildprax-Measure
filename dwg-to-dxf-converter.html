DWG to DXF Converter Guide

Buildprax Measure Pro opens DXF files directly in the drawing viewer. If your drawing is in DWG format, convert it to DXF first, then open it in Buildprax.

We recommend the free ODA File Converter for DWG to DXF conversion on both Windows and macOS.

Step 1 - Download and Install ODA File Converter

  1. Open the official ODA download page.
  2. Download the installer for your platform (Windows or macOS).
  3. Install the application with default settings.
Download ODA File Converter Back to Download Page

Step 2 - Convert DWG to DXF

  1. Open ODA File Converter.
  2. Select your input folder containing DWG files.
  3. Select an output folder.
  4. Set output type to DXF (recommended version: ACAD 2018 or similar).
  5. Run conversion.

Step 3 - Open DXF in Buildprax

  1. Open Buildprax Measure Pro.
  2. Go to the DXF/DWG viewer tab.
  3. Click Open Drawing (DXF/DWG).
  4. Select your converted DXF file.
Important: ODA File Converter is a third-party tool and is not bundled with Buildprax. Please review their terms on their official website.
